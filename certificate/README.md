# Certificate notes

This directory is reserved for certificate metadata and related review notes.

The certificate currently consists of the writeup in
`notes/turan_42a_improved_upper_bound.md` and the exact rational-interval
verifier in `scripts/verify_42a_certificate.py`.

The current package records a proposed two-block certificate for

```text
limsup_{n -> infinity} R_n <= 0.6906538.
```

AI assistance was used to help format the writeup and prepare verification
scripts. The mathematical claim, references, constants, and computations should
be independently reviewed before any public submission.
